4. Контрольні запитання¶

1. Що таке pip і conda та чим вони відрізняються між собою?

  • pip — це стандартний менеджер пакетів для Python.
  • conda — менеджер пакетів і середовищ, який може працювати не лише з Python, а й з іншими мовами (наприклад, R).
  • conda також дозволяє створювати ізольовані середовища (conda create), тоді як pip — тільки встановлює пакети.

2. Як виглядатиме виконання пунктів 1–3 індивідуального завдання за допомогою Virtualenv?

4. Написати процедуру на Python сортування масиву методом бульбашки.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

print(bubble_sort([5, 3, 8, 4, 2]))
